僕が使っているSublime Text 2のキーバインドと設定です。

ファイルを直接適切なフォルダにコピーするか、テキストをコピー&ペーストしてください。

フォルダに直接コピーする場合のパスは
C:\Users\[User_name]\AppData\Roaming\Sublime Text 2\Packages\User
のフォルダにダウンロードしたファイルをコピーして、上書きしてください。


心配な方は、Sublime Text 2の[Preferences] > [Settings - User]と[Key Bindings - User]をテキストを直接コピーすることをおすすめします。


設定に対して対応するファイル名は
[Settings - User] = Preferences.sublime-settings
[Key Bindings - User] = Default (Windows).sublime-keymap


※自己責任ですべての作業を行ってください。僕は責任を取りません。
